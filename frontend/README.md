AI Sentry Turret – Frontend

This is the frontend dashboard for the AI-Powered Sentry Turret project.

It provides a visual interface to simulate and display:

System state (scanning, tracking, etc.)
Detection metrics (FPS, confidence)
Servo positions (yaw, pitch)
Human body points (head and chest)

How to Run

1. Install dependencies
   npm install
2. Start the development server
   npm run dev
3. Open in browser
   http://localhost:5173
